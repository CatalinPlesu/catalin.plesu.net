Mandelbrot Fractal Visualizer

Interactive fractal explorer written in Rust using Dear ImGui for the UI panel. Multithreaded CPU rendering — no shaders, no GPU, just careful threading to keep the interaction responsive. Tab cycles color schemes, scroll wheel zooms, click re-centers the world, right-click produces near-Julia sets, spacebar toggles the animation sweep. Dear ImGui's immediate-mode model made building the settings panel straightforward and feels like a proper tool rather than a toy demo.

ErgoType master's thesis

Genetic algorithm for keyboard layout optimization — my master's thesis. Evolves layouts against a configurable ergonomic fitness function, accounting for finger travel distance, same-hand alternation, and corpus frequency. The algorithm works and the results are measurable (~10–15% improvement over Colemak on the tested corpus), though not yet enough to justify switching for most users. Uses UV, a modern Rust-written Python package manager, for reproducible environment management.

Canti IoT bachelor's thesis

Portable environmental monitoring system built as a bachelor's thesis. Canti 1 is a custom open-hardware device — ESP32-C3 with a BME680 and additional sensors measuring temperature, humidity, air quality, light, and noise. Canti Hub is the companion Flutter mobile app that receives data, stores it long-term, and can alert the user when any parameter drifts out of range. Firmware written in C++ with Arduino; more design patterns ended up in the embedded side than in the app. Fully open source and open hardware on GitHub.
