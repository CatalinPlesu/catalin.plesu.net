Mandelbrot Fractal Visualizer

Interactive fractal explorer written in Rust using Dear ImGui for the UI panel. Multithreaded CPU rendering. Tab cycles color schemes, scroll wheel zooms, click re-centers the world, right-click produces near-Julia sets, spacebar toggles the animation sweep.

Home Lab (ongoing)

Two-node Proxmox cluster — one always-on laptop node, one desktop brought up via Wake-on-LAN on demand. AdGuard handles local DNS and network-wide ad/malicious domain blocking. Nginx Proxy Manager provides SSL termination and reverse proxying. An SSH tunnel through a Hetzner VPS bypasses CGNAT, exposing local dev environments publicly without port forwarding. Rootless Podman for containers; Grafana + Alloy for observability across the whole stack. Working toward Terraform / K3s or NixOS.

Moldova Job Market Scraper (redesigning)

ETL pipeline scraping Moldovan job boards — built to track numbers: hiring trends, stack frequencies, salary ranges across tech and adjacent fields. Systemd timers run the scraper on a schedule; Pandas handles transformation and structure. The original version loaded a 100+ MB SQLite file into the browser, which was a fatal design flaw. Currently being rebuilt with a proper backend so the data is actually queryable.

ErgoType (master's thesis)

Genetic algorithm for keyboard layout optimization — my master's thesis. Evolves layouts against a configurable ergonomic fitness function, accounting for finger travel distance, same-hand alternation, and corpus frequency. The algorithm works and the results are measurable (~10–15% improvement over Asset on the tested corpus). Uses UV for reproducible environment management.

Canti IoT (bachelor's thesis)

Portable environmental monitoring system. Canti 1 is a custom open-hardware device — ESP32-C3 with a BME680 and additional sensors measuring temperature, humidity, air quality, light, and noise. Canti Hub is the companion Flutter mobile app that receives data, stores it long-term, and alerts when any parameter drifts out of range. Firmware written in C++ with Arduino. Fully open source and open hardware.

8-bit CPU

Designed a working 8-bit CPU in Logisim from logic gates up — ALU, register file, control unit, memory bus. Then wrote a Python assembler to compile human-readable assembly into machine code for the custom ISA. It runs, but there's no programming mechanism yet; you have to manually load the program into its integrated RAM before execution.
